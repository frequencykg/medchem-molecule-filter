class medchem_filter.filters.PropertyFilter(logp_range=None, tpsa_range=None, mw_range=None, hbd_range=None, hba_range=None, rotatable_bonds_max=None, aromatic_rings_range=None)[source]

Filter molecules based on calculated molecular properties.

Supports filtering by logP, TPSA, HBD, HBA, molecular weight, etc.

__init__(logp_range=None, tpsa_range=None, mw_range=None, hbd_range=None, hba_range=None, rotatable_bonds_max=None, aromatic_rings_range=None)[source]

Initialize property filter with ranges.

Parameters:

  • logp_range (Optional[Tuple[float, float]]) – (min, max) tuple for logP

  • tpsa_range (Optional[Tuple[float, float]]) – (min, max) tuple for TPSA in Ų

  • mw_range (Optional[Tuple[float, float]]) – (min, max) tuple for molecular weight in g/mol

  • hbd_range (Optional[Tuple[int, int]]) – (min, max) tuple for number of H-bond donors

  • hba_range (Optional[Tuple[int, int]]) – (min, max) tuple for number of H-bond acceptors

  • rotatable_bonds_max (Optional[int]) – Maximum number of rotatable bonds

  • aromatic_rings_range (Optional[Tuple[int, int]]) – (min, max) tuple for number of aromatic rings

Filter Patterns

PAINS Patterns

PAINS (Pan-Assay Interference Compounds) filter patterns.

These patterns identify compounds that frequently show up as hits in high-throughput screens but are likely to be false positives due to various interference mechanisms.

Reactive Patterns

Reactive group filter patterns.

These patterns identify functional groups that are chemically reactive and may cause issues in biological assays or drug development.

Heterocycle Patterns

Heterocycle filter patterns.

These patterns identify common heterocyclic scaffolds in medicinal chemistry.
